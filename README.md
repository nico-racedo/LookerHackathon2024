## Finalist – Looker Hackathon ’24 (Google Cloud)
#### Automatic LookML code generation for visual custom drilling in metrics.
-	Used a Python Colab notebook and the Looker SDK API 
-	The hack extracts parameters from a Look and automatically generates a LookML measure that contains a visualization for drilling.
